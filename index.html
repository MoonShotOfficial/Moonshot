Frequently Asked Questions

What is MoonShot?
MoonShot is a next-generation blockchain platform designed to deliver unprecedented performance, scalability, and security for decentralized applications.
How does MoonShot achieve high TPS?
MoonShot utilizes a novel consensus algorithm that combines proof-of-stake with advanced sharding techniques to achieve over 100,000 transactions per second.
Is MoonShot compatible with Ethereum?
Yes, MoonShot is EVM-compatible, allowing developers to port existing Ethereum dApps with minimal modification.
How do I get started with MoonShot?
You can start by visiting our documentation portal, downloading our SDK, or joining our developer community.
What programming languages are supported?
MoonShot currently supports Solidity, Rust, and TypeScript with more languages planned for future releases.
Are there grants available for developers?
Yes, we have a comprehensive developer grants program to support innovative projects built on MoonShot.
How secure is MoonShot?
MoonShot employs multiple security layers including quantum-resistant cryptography and rigorous consensus validation.
Can I stake MOON tokens?
Yes, staking is available for MOON token holders, providing rewards while supporting network security.
Where can I buy MOON tokens?
MOON tokens are available on major exchanges including Binance, Coinbase, and our official DEX partners.
